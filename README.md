# Ecommerce-Sales-Analysis
Excel-based E-commerce sales analysis with dashboard and insights  ✔ Make it Public
Project Overview:
This project involves cleaning, transforming, and analyzing e-commerce sales data using Excel.

Key Features:

Data cleaning using Power Query
Data integration using VLOOKUP
Descriptive statistics (mean, median, std deviation)
Pivot table analysis
Interactive dashboard with slicers

Key Insights:

Sports category generated highest revenue
Balanced payment distribution
High variability in customer spending

Tools Used:

Microsoft Excel
Power Query
Pivot Tables
